Email Classification — CA1

A machine learning project that automatically classifies customer support emails into categories using Random Forest and TF-IDF.

## Team
Saravanakumar — model architecture & training

Praveen — data preprocessing & pipeline

Takes raw customer emails from two datasets (AppGallery & Purchasing), cleans them, converts them to numbers, and trains a classifier

Result: 76% accuracy on 206 emails

## How to run

```bash
# Install dependencies
pip install -r requirements.txt

# Run the project
python main.py
```
